- `PWA 적용 확인 방법`
- `1. Service Worker 확인`
- Application 탭 > Service Workers
- http://localhost:3000/sw.js 등록 여부 확인
- 상태: "activated and is running"
- `2. Manifest 확인 `
- Application 탭 > Manifest
- 페이지 새로고침 (Ctrl+F5 또는 Cmd+Shift+R)
- Manifest 정보가 표시되어야 함
- `3. Cache Storage 확인`
- Application 탭 > Storage > Cache Storage
- offlineCache, start-url 등 캐시 확인
- `4. Network 탭에서 오프라인 테스트`
- Network 탭 열기
- Throttling에서 "Offline" 선택
- 페이지 새로고침
- 오프라인에서도 동작하면 성공
