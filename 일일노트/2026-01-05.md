- `EC2에 Nginx 설치`
- sudo apt update
- sudo apt install -y nginx
- sudo systemctl enable nginx
- sudo systemctl start nginx
- curl http://localhost - 설치 확인

- `Nginx 서비스 상태 확인`
- sudo systemctl status nginx
- `Nginx 버전 확인`
- nginx -v 
- `실행중인 포트 확인`
- sudo ss -tuln | grep -E ':(80|443)' 

- `설정 파일 생성`
- sudo nano /etc/nginx/sites-available/api.kanggyeonggu.store
- `설정 파일 확인`
- sudo cat /etc/nginx/sites-available/api.kanggyeonggu.store
-  
- `설정 문법 확인`
- sudo nginx -t
- `기본 설정 비활성화`
- sudo rm /etc/nginx/sites-enabled/default
- `새 설정 활성화`
- sudo ln -s /etc/nginx/sites-available/api.kanggyeonggu.store /etc/nginx/sites-enabled/
- `Nginx 재시작`
- sudo systemctl restart nginx
