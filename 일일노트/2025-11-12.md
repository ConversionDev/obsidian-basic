- `Join Point` 실행 가능한 지점 (메서드 호출, 예외 발생 등) → 예: public void save(User user) 호출 순간
- `Pointcut` Join Point 중 실제 관심 있는 지점만 필터링 → 예: @annotation(RequestMapping) 붙은 메서드만 포인트에서 정한 지점.
- `Advice` 실제 수행할 코드 (관심사 로직) → 예: “로깅”, “시간 측정”, “트랜잭션 시작/종료” 
- `Aspect` Pointcut + Advice 의 조합 Weaving Aspect를 실제 코드 실행 시점에 주입하는 과정 → Spring에서는 프록시(Proxy) 기반으로 Weaving 수행
