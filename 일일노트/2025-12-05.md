- 머신러닝에서 **전처리(Preprocessing)**는 _모델이 학습하기 좋은 형태로 데이터를 다듬는 과정_
- 
- 머신러닝 전처리의 주요 항목

## 1. **결측치 처리 (Missing Values)**

데이터에 빠진 값이 있을 때

- 제거(drop)
    
- 평균/중앙값/가장 흔한 값으로 대체
    
- 고급 기법(Regression, KNN Imputer 등) 사용
    

## 2. **이상치 처리 (Outliers)**

정상 범위를 벗어난 극단값

- 제거
    
- IQR, Z-Score로 탐지
    
- 로그변환 등으로 완화
    

## 3. **정규화/표준화 (Normalization/Standardization)**

값의 범위가 다르면 모델이 특정 특성에 치우쳐 학습함

- Min-Max Scaling (0~1)
    
- Standard Scaling (평균0, 분산1)
    

신경망·KNN·SVM 등에서 특히 필수.

## 4. **카테고리 변수 인코딩 (Encoding)**

문자열을 숫자로 변환

- One-Hot Encoding
    
- Label Encoding
    
- Target Encoding
    

예) “빨강/파랑/초록” → 0/1로 변환

## 5. **특성 선택 (Feature Selection)**

불필요한 컬럼 제거

- 상관관계 기반 제거
    
- 중요도(feature importance) 기반
    
- PCA 등 차원 축소
    

## 6. **데이터 분할 (Train/Test Split)**

학습용 / 검증용 / 테스트용으로 나눔  
일반적으로

`train 70 ~ 80% validation 10~15% test 10~15%`

## 7. **데이터 증강 (Data Augmentation – 이미지/텍스트)**

특히 이미지·텍스트에서 데이터 부족하면

- 이미지 → 회전, 확대, 잡음 추가
    
- 텍스트 → 단어 치환, 랜덤 삭제
