정리

- 노트북 → 먼저 push 했음
    
- 데스크탑 → pull 안 하고 작업 → push 충돌
    
- 해결 → `git pull --rebase` 후 `git push`
    
- 충돌 나면 파일 수정 후 `git add`, `git rebase --continue`
