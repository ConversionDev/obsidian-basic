## 📝 구현된 주요 기능

### 1. JWT 토큰 생성/검증 (JwtTokenProvider)

- generateToken() - 사용자 정보로 JWT 생성

- validateToken() - JWT 유효성 검증

- extractClaims() - JWT에서 사용자 정보 추출

### 2. 카카오 API 통신 (KakaoService)

- getAccessToken() - 인가 코드로 카카오 액세스 토큰 요청

- getUserInfo() - 액세스 토큰으로 사용자 정보 조회

### 3. 카카오 로그인 엔드포인트 (KakaoController)

- POST /kakao/login - 인가 코드로 로그인 처리

- GET /kakao/callback - 카카오 리다이렉트 콜백 처리

### 2. 프론트엔드 수정 필요

프론트엔드에서 카카오 로그인 버튼을 클릭하면:

1. 카카오 인증 페이지로 리다이렉트

2. 사용자 로그인 후 redirect_uri로 code 전달

3. code를 백엔드로 전송: POST /api/auth/kakao/login { "code": "..." }

4. JWT 토큰 받아서 저장
