### 에이전트 슬라이스 구조
- 5개의 에이전트 슬라이스: `agent1.ts` ~ `agent5.ts`

### 스토어 구조
```
app/store/
├── slices/
│   ├── agent1.ts ~ agent5.ts  # 5개 에이전트 슬라이스
│   ├── gpt.ts                 # GPT 슬라이스 (별도 분리)
│   ├── search.ts
│   ├── user.ts
│   └── ui.ts
```

### 주요 특징
1. 에이전트 독립성: 각 에이전트는 독립적인 슬라이스로 관리
2. GPT 분리: GPT 기능은 별도 슬라이스로 완전 분리
3. 공통 인터페이스: 모든 에이전트는 `AgentState`와 `AgentActions` 상속
4. 확장성: 각 에이전트는 고유한 액션 추가 가능 (예: `processData`, `analyzeData` 등)

### 사용 예시
- 각 에이전트는 독립적으로 사용: `useStore((state) => state.agent1)`
- GPT는 별도로 사용: `useStore((state) => state.gpt)`
- 커스텀 훅 제공: `useAgent1()`, `useAgent2()`, ..., `useGPT()`
