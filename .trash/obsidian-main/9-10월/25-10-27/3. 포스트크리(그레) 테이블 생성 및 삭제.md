csv 파일을 도커 안에 있는 컨테이너 안에 db 안에 넣어서 테이블로 만들기
포스트그리 =db에서 쓰는 무언가

- PostgreSQL 컨테이너가 실행 중인지 확인
	docker ps

1. csv 파일을 컨테이너로 복사
docker cp "C:\Users\hi\Documents\project\api-server\src\main\resources\static\csv\greenhouse_data.csv" esgaida:/tmp/greenhouse_data.csv

2. tmp 라는 폴더가 있는지 확인
	docker exec esgaida ls -id /tmp

3. 도커 입장
	docker exec -it esgaida bash : 도커 입장

4. 데이터베이스 입장
	psql -U esgaida -d esgaidadb : 데이터베이스 입장

	** 한 번에 도커-데이터베이스로 들어가기
		docker exec -it esgaida psql -U esgaida -d esgaidadb

5. db에 같은 이름으로 이미 테이블이 존재할 때 삭제하는 법
	DROP TABLE IF EXISTS test(테이블이름);
DROP TABLE IF EXISTS energys;

6. 테이블 만들기
CREATE TABLE test (corporate_name TEXT, supervising_authority TEXT, designation_type TEXT, industry TEXT, ghge NUMERIC, energy NUMERIC, verification TEXT);

7.  데이터를 PostgreSQL에 Import
COPY test(corporate_name, supervising_authority, designation_type, industry, ghge, energy, verification)
FROM '/tmp/greenhouse_data.csv'
DELIMITER ','
CSV HEADER
ENCODING 'UTF8';

*7번이랑 같은 작업인데, ps 환경에서 입력할 경우
docker exec esgaida psql -U esgaida -d esgaidadb -c "COPY test(corporate_name, supervising_authority, designation_type, industry, ghge, energy, verification) FROM '/tmp/test.csv' With(FORMAT csv, HEADER true, DELIMITER ',');"
