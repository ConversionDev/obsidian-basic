- NL : natural language / ML : machine language
- 인터페이스 : 주고 받는 것 (대화)
- 소스와 리소스
    - 소스 : 컴파일 단계 ,자바 컴파일러가 `.class`로 변환
    - 리소스 : Run 단계, 그대로 복사되어 배포 (가공 안 함)
- 인터넷 주소창에 [localhost:8080](http://localhost:8080) 을 입력하는 것은 컴파일링 한다는 것을 의미함.
- html은 인터프리팅이니까 로컬에 바로 안뜸, 도메인에 올리기 위해서는 컴파일을 해야 함. 즉, 자바에 연동해야함, 그래들을 해야 함 (인터프리팅만 하면 그 모니터에서만 보임)
- ; 은 is를 의미함.
- class path: View가 Model로 넘어가는 길 = 클래스로 이동하는 길
- MVC (Model - View - Controller)
    - Model: 데이터와 비즈니스 로직을 담당
    - View: 사용자에게 보여주는 화면(UI)를 담당
    - Controller: 사용자의 요청을 받아서 처리하고, Model과 View를 연결해줌. 즉, 둘 Model과 View의 교집함
