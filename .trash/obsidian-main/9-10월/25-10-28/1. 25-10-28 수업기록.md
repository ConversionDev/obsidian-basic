| 명령어                                              | 뜻                                                         |
| ------------------------------------------------ | --------------------------------------------------------- |
| select * from test;                              | test라는 이름의 테이블의 모든 내용(\*)을 보여줘                            |
| select * from stadium where stadium_id = 'D03' ; | 테이블 이름이 stadium 인 곳에서, stadium_id가 D03 인 경우의 모든 행 값을 보여줘. |
| select * from stadium limit 5                    | 테이블 이름이 stadium 인 곳에서, 첫 5줄만 보여줘                          |

- relations 는 table이고, relationship은 table 들을 연결해 주는 다리임
	- 1번은 부모, 2번은 자식임. 2번에서 다리 세개짜리를 크로우핏 이라고 부르며, 2번에서 동그라미는 null 값도 허용하겠다는 의미임
	- 1-2번 관계는 1:多 관계이고 /  1-3번 관계는 1:1 관계임
	- 자식은 부모의 PK(primary key)를 FK로 갖는다 (=자식은 부모의 DNA를 갖는다)
	- 부모는 자식의 DNA를 가질 수 없다.
