7. 딥러닝의 개요
	- 퍼셉트론(Perceptron): 지각하는 기계 (퍼셉:지각, 트론:기계)
	- 이미테이션: 현실 모방
	- GPU는 다 가상이다
	- 갔다가 다시 오는 과정에서 피드백을 받으면서 값이 강화됨.
		- w는 가중치, y는 인공뉴런
		- 가중치 값이 커질수록 중요함을 의미
		- 가중치 되기 전에 편차
			![[Pasted image 20251212150839.png]]
	-  인공 신경망 Artificial Neural Network ANN
			![[Pasted image 20251212152006.png]]
		- = 퍼셉트론 여러개
		- = 다층 퍼셉트론
		- = 층이 많아지면서 깊어지니까 딥러닝이라고 부르게 된 것 (한 층만 하면 머신러닝)
		- DS 중 여러개의 피처 중에 뭐가 중요하고 결과가 무엇인지를 자동으로 훈련된 모델을 쓸거야
		- 활성화 함수(Activation Function) = 자동 DROP
			- 강아지냐 고양이냐 구분시키는 과정에서 사진을 벡터화해서 한 줄로 만들지. 배경에 대한 부분은 필요 없으니까 그 부분을 삭제하는 것
			- 층을 넘어갈 때마다 계속 지울건지우고 남길건 남기는 것
			- 옛날에는 0과 1로만 나뉘는 아래와 같은 상태였기 때문에 지웠던 피처를 다시 되돌리기 어려웠음 -> 그 상태에서 초록색 구간처럼 되면서 상대적으로 유연해진거지.
				- 예를들어 강아지 고양이 사진에서 하얀 배경을 지운다고 했을 때 계단함수였으면 하얀 눈동자 부분까지 소실되었던거임. 그래서 시그모이드?함수? 밑에 초록색 부분처럼 하면 그걸 막아주는 것임
				![[Pasted image 20251212154835.png]]![[Pasted image 20251212155022.png]]
		- 행렬 곱으로 이해하는 신경망
			- model = Sequential()   --> 시퀀스: 여러 층을 이동하면서 쫙 퍼졌다가 마지막에 하나로 합쳐지는 것
			- 3개의 입력과 2개의 출력
				- model.add(Dense(2, input_dim=3, activation='softmax'))
					- Dense: 빽빽한 것 <-> Sparse: 0값이 많은
					- 소프트멕스: 엑티베이션 층 마다를 부드럽게 연결하겠다는 의미
			- 딥러닝의 식 y = wx +b  (b가 편차)
					![[Pasted image 20251212160527.png]]
		- 딥러닝의 학습 방법
			- 손실 함수
				- 예측값과 실제값의 차이를 줄이면서 고쳐나가는 것 = 손실함수
			- 역전파
				- 예측값이 틀렸을 때 되돌아가서 수정하는 것. 가충치를 변경
			- 과적합(Overfitting) 막는 방법
				- 데이터 양 늘리기, 모델 복잡도 줄이기, 가중치 규제 적용하기, 드롭하기 등
		- 케라스 훑어보기
			- 원핫 인코딩이 여러개면 워드 임베딩 (밀집 벡터)
			- 근데 케라스는 텐서플로우꺼라 우리가 못 씀
