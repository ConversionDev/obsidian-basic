- 브랜치 처음 연결할 때
PS C:\Users\hi\Documents\project\front\feature> git clone https://github.com/ESGAIFixr/aifixr.site.git .

PS C:\Users\hi\Documents\project\front\feature> git fetch

PS C:\Users\hi\Documents\project\front\feature> git checkout feature-ys

PS C:\Users\hi\Documents\project\front\feature> git branch --show-current
feature-ys


----
- 내 브렌치 데브에 올리 때 (데브 터미널에서)
PS C:\Users\hi\Documents\project\front\dev> git checkout dev
Already on 'dev'

PS C:\Users\hi\Documents\project\front\dev> git pull origin dev

PS C:\Users\hi\Documents\project\front\dev> git pull origin feature-ys (브렌치 3명꺼 가져와서 머지할거면 세명 브랜치 이름을 나열하기(컴마로 나열하는지는 찾아봐야 함) )

PS C:\Users\hi\Documents\project\front\dev> git merge feature-ys (여기도 세명 꺼) 
위에꺼 한 뒤에 (푸쉬 전에) 데브 커서 열어서 오류 있는지 확인. 그 후에 데브 푸쉬 할 것

PS C:\Users\hi\Documents\project\front\dev> git push origin dev
